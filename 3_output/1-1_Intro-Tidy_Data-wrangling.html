INTRO TO TIDYVERSE: DATA WRANGLING

Tidyverse is a collection of data science tools within R for transforming and visualizing data. It is a powerful and popular approach for exploring data.

1.1 The Gapminder dataset

Gapminder is a well-known dataset that tracks economic and social indicators of countries over time.

The dataset is stored in a tibble which is a special type of data frame. It has 1704 observations (rows), 6 variables (columns). It is important in an analysis to understand what each observation, or row, represents. Here it represents a unique pair of a country and a year.
For each combination of a country and a year, the dataset contains several variables, or columns, describing the country’s demographics, population, life expectancy, GDP per capita etc. Each variable is of one consistent data type: numerical (life expectancy, population), and categorical (country, continent).

1.2 Filter

The dplyr package in tidyverse uses verbs that are atomic steps to transform data.

The filter verb extracts particular observations based on a particular condition. Filtering is a common first step in data analysis.
Every time you apply a verb in dplyr, use a pipe %>% that takes whatever before it and feeds it to the next step.

For example, filter the gapminder data set for the year 1957:

gapminder %>% 
  filter(year == 1957) # "==" shows logical equal.

1.3 Mutate

The mutate verb allows to change an existing variable or adding a new one based on the other ones.

For example, calculate population per million:

gapminder %>%
  mutate(pop = pop / 1000000) # the value in pop column is much smaller now.
## # A tibble: 1,704 x 6

##  2 Afghanistan Asia       1957    30.3  9.24      821.
##  3 Afghanistan Asia       1962    32.0 10.3       853.
